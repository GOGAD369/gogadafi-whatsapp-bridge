📖 Instructions

🚀 How This Dashboard Works

1
All Chats — View Conversations
Click "All Chats" in sidebar to see all customer conversations. Click any customer to view their chat history.
2
Manual Reply — Override Bot
Open a chat → type in the reply box → press Enter or click Send. Your reply goes directly to the customer via WhatsApp. Bot continues after your reply.
3
Outbound Message — Contact New Customers
Click "Outbound" in sidebar → enter phone number + message → send. Use this to proactively contact customers.
4
Contact Tags — Label Customers
In the Contacts page, click the tag pill on any row to cycle it between "Hot Lead", "Order Placed" and "New Customer".
5
Export Data
Use the Export button on the Contacts or Message Logs page to download data as CSV. Open in Excel for analysis.

🔧 Changing Dashboard Password

1
Go to Railway
railway.app → your project → Variables
2
Edit DASHBOARD_PASSWORD
Change the value to your new password → Update Variables → auto-redeploy
DASHBOARD_PASSWORD="your-new-password"
